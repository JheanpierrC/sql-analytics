📊 Sales Analytics
📌 Descripción
  Proyecto práctico usando la base de datos AdventureWorks2025 para analizar ventas por cliente utilizando:
      * CTE (Common Table Expressions)
      * Funciones de agregación
      * Window Functions
      * RANK()
      * Running Totals
      
🎯 Objetivo del análisis
  Para cada pedido:
    * Mostrar el total del pedido
    * Calcular el total acumulado por cliente
    * Identificar el pedido más grande de cada cliente
    
🛠️ Tecnologías utilizadas
    * SQL Server
    * AdventureWorks2025
    * SQL Server Management Studio (SSMS)
    
🧠 Conceptos aplicados
    * GROUP BY
    * INNER JOIN
    * SUM() OVER()
    * RANK() OVER()
    * PARTITION BY
    * ROWS BETWEEN
    
🚀 Resultado esperado
  El script devuelve:
    * SalesOrderID
    * NombreCliente
    * TotalPedido
    * TotalAcumuladoCliente
    * RankingDentroCliente
    * EsPedidoMasGrande
